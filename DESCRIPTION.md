# AGI — Agentic Generative Interface

AGI (Agentic Generative Interface) is a native iOS application that applies Generative UI co-creation to real-world device interaction. Instead of relying on fixed screens or static tutorials, AGI creates a contextual interface at runtime from the camera view, the user's goal, and scene understanding.

The user points the phone at an unfamiliar physical interface (for example, an intercom, thermostat, or control panel). AGI analyzes the scene, proposes likely intents, and generates an interactive UI overlay with step-by-step guidance tailored to that exact object and moment. The user and AI co-create the experience: the user provides intent and feedback, while the AI generates and adapts the interface in real time.

To make exploration feel engaging rather than utilitarian, we layered in gamification inspired by RPG mechanics: a perception stat system, collectible artifact cards, skill check badges, and an AI companion narrator with a distinct literary personality. Custom sound effects and adaptive background music turn each interaction into something closer to a game quest than a manual lookup.

We also built a Saved Skills mechanism: useful generated artifacts can be bookmarked and recalled later. If the environment changes, AGI can re-generate the skill for the new scene, keeping guidance relevant instead of replaying stale output.

In short, we built a system where interfaces are not pre-authored once, but dynamically co-created with AI to help people confidently — and enjoyably — interact with the physical world.
